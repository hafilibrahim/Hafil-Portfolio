EnerSafe — LT Transmission Line Fault Detection

A real-time fault detection and isolation system that uses voltage-current analysis to identify transmission line breakage and hazardous live-wire conditions. On fault detection, the system automatically triggers transformer isolation and transmits alerts via LoRa to electricity board systems — protecting public safety and cutting response time dramatically.
